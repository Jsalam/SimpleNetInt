Add Categories of edges

Description

Categories are qualitative aspect that characterize links. Classes are also named kinds or classes. For instance the "friend" class represents the friendship between nodes A and B. Another link of the "classmates" class, can also connect the same nodes. In Simple Networks there is a link for every kind of relationship between nodes.

Enter the categories you want to use in this network separate by comas. For example: friend,classmate,partner,relative
